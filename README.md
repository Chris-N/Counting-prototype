# Counting-prototype
Second game. Control swinging a bat to hit all the balls launched at you.

## Game Instructions
- Press "Spacebar" to swing the ball
- Press "a" to begin the game.
- Press "p" to begin practice mode
- Press "?" for more instructions in game
